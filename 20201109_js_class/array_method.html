요소찾기

item의 위치값을 찾는다/start위치부터 item의 위치값을 찾는다.
indexOf(Item), indexOf(Item, start)
lastIndexOf(Item), lastIndexOf(Item,start)

배열 뒤 추가삭제

push(item) : 배열의 마지막 요소에 item을 추가
pop() : 배열의 마지막 요소를 제거
즉, 마지막 요소에 추가 / 삭제

배열 앞 추가삭제

반대 개념으로 shift() 는 맨 앞값 제거
unshift() 는 맨 앞에 값을 추가

Splice-수정(삭제/추가)

splice(index, n) - index 위치에서 n개를 삭제
splice(index, n, a,b,c) - index위치에서 n개를 삭제하고 a,b,c를 추가
plice(index, 0, a,b,c)- index위치에서 삭제하지 않고 a,b,c를 추가
